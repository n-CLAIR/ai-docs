Usage of resources
-------------------

    session의 자원 사용량을 확인하는 방법은 2가지가 있습니다.

    1) session 화면에서 눈모양을 선택

    .. figure:: ../../static/nsml_web/nsml_web_click_eye.png

    2) 메인 화면에서 눈모양을 선택

    .. figure:: ../../static/nsml_web/nsml_web_main_click_eye.png

    해당 session에 대해 눈모양을 클릭하면 실행한 session 의 GPU, CPU, memory 사용량을 그래프로 볼 수 있습니다.

    .. figure:: ../../static/nsml_web/nsml_web_usage_of_gpus.png

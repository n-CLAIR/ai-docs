NSML Constants
--------------

    .. py:attribute:: nsml.IS_ON_NSML

        NSML server 에서 code 가 실행되는 상태면 True를 출력합니다.

    .. py:attribute:: nsml.HAS_DATASET

        session 이 dataset 을 가지고 있으면 True를 출력합니다.

    .. py:attribute:: nsml.DATASET_PATH

        :ref:`nsml run -d DATASET<nsml run>` 으로 실행시켰을 때 dataset 의 path를 출력합니다.

    .. py:attribute:: nsml.DATASET_NAME

        :ref:`nsml run -d DATASET<nsml run>` 으로 실행시켰을 때 dataset 의 이름입니다.

Overview
---------

    :ref:`prepare_a_session_environment<prepare session environment>` 페이지에서는 NSML에서 진행될 코드 실행 환경을 설정하는 방법을 알아보겠습니다. 반드시 확인하고 넘어가시기 바랍니다.

    :ref:`run a session<nsml run>` 페이지는 NSML에서 가장 많이 쓰이는 명령어를 소개합니다. session을 시작하는 명령어이며 다양한 옵션을 통해 session을 실행할 수 있습니다. GPU, CPU 등 자원의 양을 옵션으로 설정할 수 있는 중요한 파트이기 때문에 반드시 확인하고 넘어가시기 바랍니다.

    :ref:`submit_a_session<nsml submit>` 페이지에서는 leaderboard에 모델을 올리는 명령어를 소개합니다. 해커톤에서 꼭 사용해야 하는 명령어입니다.

    나머지 페이지들은 session과 model들을 다운로드받고, 조회하고, 삭제하는 내용이 담겨있습니다.

    다음 페이지에서는 session 환경 설정( :ref:`prepare_a_session_environment<prepare session environment>` )에 대해 알아보도록 하겠습니다.

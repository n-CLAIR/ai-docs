Delete a model¶

Usage:

nsml model rm SESSION_NAME CHECKPOINT
Description:

Delete one or several models. Type number of CHECKPOINT.

Example:
nsml model rm nsmlteam/mnist/4 24

nsml model rm nsmlteam/mnist/4 "24"

Show leaderboard of dataset¶

Usage:

nsml dataset board Options DATASET_NAME
Description:

It shows leaderboard based on dataset uploaded to NSML.

Example:
Basic command¶
nsml dataset board mnist

nsml dataset board -j mnist
Options:
-j, --json-return

It shows in json format.

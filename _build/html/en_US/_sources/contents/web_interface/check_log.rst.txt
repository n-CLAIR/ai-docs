.. _check log:

Check log
---------

    :ref:`nsml run<nsml run>` 으로 실행한 세션로그를 web 에서 확인할 수 있습니다.

    첫페이지 데이터셋 목록에서 왼쪽에 데이터셋 목록이 보이고,

    데이터셋 이름 왼쪽의 화살표를 클릭합니다.

    .. figure:: ../../static/nsml_web/nsml_web_selected_dataset_point.png

    화살표를 클릭하면 session list 가 나옵니다. 그중에 하나를 클릭해서 session log 를 확인할 수 있습니다.

    .. figure:: ../../static/nsml_web/nsml_web_selected_session_log.png

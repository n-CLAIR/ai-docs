Pull sessions¶

Usage:

nsml pull Options SESSION_NAME PATH
Description:

nsml run 하였을 때 upload 된 session 파일들을 local machine 으로 가져옵니다.

Example:
nsml pull -v nsmlteam/mnist/4 ./
Options:
-v, --verbose

상세한 진행과정을 표시합니다.

-f, --force

로컬 path 에 folder 가 있으면 지우고 덮어씁니다.

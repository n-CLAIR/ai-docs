.. _nsml dataset search:

Search dataset
--------------

    .. <br />

    Usage:
        +-----------------------------------------+
        | **nsml dataset search** *DATASET_NAME*  |
        +-----------------------------------------+

    .. <br />

    Description:
        NSML에 등록된 데이터셋을 이름으로 찾아줍니다.

    .. <br />
    .. <br />

    Example:
        .. code-block:: console

            nsml dataset search mnist
